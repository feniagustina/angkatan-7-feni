PENGALAMAN
Hallo,semuanya saya ingin berbagi cerita sama kalian
semua tentang pengalaman saya yang pernah mendapat
juara 1 lomba pidato bahasa indonesia.Saat itu saya masih
kelas 4/5 MI,kemudian saya dipanggil sama ibu guru saya
untuk menemui nya dan disuruh ikut lomba pidato di ajang
porseni untuk mewakili sekolahan.Pada saat itu saya berusaha
menolak karena itu pertama kali saya ikut lomba,tetapi
guru saya tetap percaya sama saya kalau saya itu pasti
bisa.Dan mau gamau saya tetep ikut di lomba tersebut.Setiap
hari saya selalu latihan ke rumah guru saya,setiap hari saya
menghafalkan teks pidato tersebut.Saya latihan pidato itu butuh
waktu dua minggu.Pertama latihan,saya di latih hanya di dalam
rumah nya guru saya,tetapi pada saat mendekati hari H saya
disuruh latihan di luar,disitu saya malu karena dilihatin banyak
orang,dan saya gamau.Terus guru saya bilang gini “gapapa buat
nguji mentalnya kamu biar pas diperlombaan kamu sudah
terbiasa”.Dan ya saya latihan diluar.Pas hari H tiba,sekitar pukul
07.00 saya berangkat ke tempat perlombaan tersebut,dan pas
sayaa sampai di tempatnya saya merasa gugup.Kemudian saya
masuk ke ruangan nya dan nunggu giliran saya.Nah pas peserta
sebelumya saya itu sudah jago banget pidatonya,saya langsung
inscure takut saya kalah dalam perlombaan itu.Tetapi tak pikir lagi kalau kalah menang dalam perlombaan itu sudah menjadi hal yang biasa.Singkat cerita saya sudah selesai pidato dan nunggu hasil pengumuman,saat dibacakan hasiil pengumuman nya saya sudah pesimis.Dan ternyata saya yang mendapat juara 1 dalam perlombaan tersebut dan melanjutkan ke tingkat KAB.
